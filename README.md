# Корпоративный сайт «Автомобильный завод»

Веб-сайт разработан в рамках учебной практики с использованием языка программирования Python и фреймворка Flask.  
Проект реализован без использования систем управления контентом (CMS) и соответствует требованиям задания по практике.

---

## 📌 Назначение проекта

Сайт предназначен для представления информации об автомобильном заводе, его деятельности, производстве, новостях, а также для взаимодействия с пользователями через форму обратной связи и систему учётных записей.

---

## 🛠 Используемые технологии

- Python 3.x  
- Flask  
- Flask-SQLAlchemy  
- SQLite  
- HTML5  
- CSS3  
- JavaScript  

CMS при разработке **не использовалась**.

## 🌐 Реализованные разделы сайта

- Главная страница  
- О заводе (не менее 5 статей)  
- Производство  
- Новости  
- Контакты  
- Поиск по сайту  
- Личный кабинет пользователя  
- Административная панель  
- Карта сайта  
- Страница 404  

Страница карты сайта и страница 404 **не учитываются** при подсчёте общего количества страниц.

---

## 👤 Пользователи и роли

В системе реализованы следующие роли:

- **Пользователь**
  - просмотр контента
  - поиск по сайту
  - отправка сообщений через форму обратной связи

- **Администратор**
  - добавление и управление статьями
  - добавление новостей
  - просмотр пользователей
  - управление ролями

Реализованы сценарии:
- самостоятельной регистрации пользователя
- входа в систему
- выхода из системы

---

## 🎨 Оформление и доступность

Разработаны два стиля оформления сайта:

1. **Стандартный стиль**
   - корпоративная цветовая схема
   - адаптивная вёрстка
   - удобная типографика

2. **Версия для слабовидящих**
   - высокий контраст
   - увеличенный шрифт
   - простая структура
   - соответствует требованиям ГОСТ Р 52872-2012

Переключение между стилями осуществляется с помощью JavaScript.

## 🔍 Поиск по сайту

Реализован поиск по:
- статьям
- новостям

Поиск работает по заголовкам и тексту материалов.

## 📬 Обратная связь

На странице «Контакты» реализована форма обратной связи.  
Сообщения пользователей сохраняются в базе данных.

## 🚀 Запуск проекта локально

1. Клонировать репозиторий:
git clone <ссылка_на_репозиторий>


2. Установить зависимости:
pip install -r requirements.txt


3. Запустить приложение:
python main.py


4. Открыть в браузере:
http://127.0.0.1:5000
